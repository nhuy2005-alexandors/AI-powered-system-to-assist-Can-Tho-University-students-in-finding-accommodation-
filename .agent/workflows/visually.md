# Quy trình Trực quan hóa Hệ thống (Professional Diagramming)
**Trigger**: `/visually`, `/diagram`, `/uml`

## Mục tiêu
Thiết kế sơ đồ hệ thống chuyên nghiệp phục vụ cho Software Architecture và Design Documentation.

## Các bước thực hiện
1. Nhận yêu cầu vẽ sơ đồ (Use Case, Sequence, Class, Architecture, v.v.).
2. Load kỹ năng `system-architect` từ `.agent/skills/system-architect/SKILL.md`.
3. Sử dụng hệ thống hiển thị Artifact của chính IDE (chuẩn Mermaid Markdown) để vẽ trực tiếp trên màn hình chat. Khuyến khích tạo file Artifact riêng biệt để người dùng dễ xem và copy.
4. (Tùy chọn) Nếu người dùng yêu cầu xuất file PNG/SVG, dùng kỹ năng Data Exporter để tự cài `mermaid-cli` và xuất file vật lý.
