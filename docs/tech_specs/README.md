# Tech Specs

Mỗi feature mới → 1 file `<feature>.md` ở đây TRƯỚC khi code.

Template:
- **Mục tiêu**: 1-2 câu feature làm gì.
- **Yêu cầu**: checklist [ ] các điều kiện phải đạt.
- **Ràng buộc**: giới hạn kỹ thuật/bảo mật bắt buộc.
- **Quyết định**: chọn gì, vì sao (chống hỏi lại).
- **Ngoài phạm vi**: cái KHÔNG làm ở phiên này.
