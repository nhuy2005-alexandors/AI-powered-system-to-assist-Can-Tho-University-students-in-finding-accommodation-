# Tài liệu — Trọ CTU

## Khung làm việc (đọc trước)
- `CHECKPOINT.md` — trạng thái hiện tại (Đã xong / Đang làm / Tiếp theo / Gotcha)
- `DECISIONS.md` — quyết định kiến trúc + lý do (ADR)
- `GOTCHAS.md` — bẫy kỹ thuật đã dính

## Thư mục
- `specs/` — thiết kế TRƯỚC khi code (mỗi feature 1 file)
- `tech_specs/` — as-built SAU khi code (module đã build, cho human đọc lại)
- `planning/` — SRS, Roadmap, Sprint Plan, phân tích nghiệp vụ/kỹ thuật/use-case (định hướng đề tài)
- `reference/` — API spec, Data Dictionary, Test Plan, Deployment, handoff
- `de-tai/` — hồ sơ đề tài NCKH (thuyết minh, hợp đồng, dự toán)
