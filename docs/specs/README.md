# Specs (thiết kế TRƯỚC khi code)

Mỗi feature → 1 file `<feature>.md` TRƯỚC khi code:
- **Mục tiêu**: feature làm gì (1-2 câu).
- **Yêu cầu**: checklist [ ] điều kiện phải đạt.
- **Ràng buộc**: giới hạn kỹ thuật/bảo mật.
- **Quyết định**: chọn gì, vì sao.
- **Ngoài phạm vi**: cái KHÔNG làm phiên này.

> As-built (SAU khi code) nằm ở `docs/tech_specs/`.
